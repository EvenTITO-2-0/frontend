# Eventito frontend

## Install the dependencies

```sh
npm install
```

## Run the app

```sh
npm run dev
```

## Run the linter

Install dependencies

```sh
npm install --save-dev prettier eslint eslint-config-prettier eslint-plugin-react @typescript-eslint/parser @typescript-eslint/eslint-plugin husky lint-staged
npm install
```

Run formatter

```
npm run format
```
